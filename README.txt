== Description ==

Connecting Element Content API wordpress integration

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload ce-capi` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
